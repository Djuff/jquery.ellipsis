ellipsis.js by bebraw

Ellipsis plugin for jQuery

License

MIT

Dependencies

jQuery (1.something)

Installation

Copy jquery.ellipsis.js to some appropriate place, link to it at your HTML, use 

$(selector).ellipsis(opts);
Check out opts and markup at source.

Download

You can also clone the project with Git by running: 

$ git clone git://github.com/bebraw/ellipsis.js

Get the source code on GitHub : bebraw/ellipsis.js
